# Проект: Место

Проект - это сайт Mesto,с адаптивом для различных устройств.
Реализована возможность изменения имени профиля и описания, открытие и закрытие формы по клику на определенные элементы.

## Используемые технологии
* HTML / CSS / BEM
* JavaScript
